## Structure

```
.
├── packages
    ├── client             # Web application, wrote with React
    ├── server             # GraphQL server used by clients
```

## Working on teamy

```
yarn                 # Install dependencies
yarn start           # Starts : Postgres, Primsa Client, Server, Client
```
